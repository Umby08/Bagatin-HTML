# Bagatin-HTML
Progetto di HTML/CSS: pagina wiki sulla serie TV americana "Bones". 

Il sito è composto da più pagine (home, personaggi, produzione, stagioni e serial killer); ogni pagina ha un CSS ad hoc, oltre che un secondo CSS - general.css - che definisce le strutture comuni a tutte le pagine (header, footer, etc).

La navigazione tra pagine è permessa da una sidebar a scomparsa e il pulsante "back to top" permette all'utente di tornare in cima alla pagina senza scorrerla tutta. Entrambe queste funzionalità sono state realizzate senza javascript. 

I CSS delle varie pagine sono strutturati secondo la metodologia BEM e il sito è responsive (in grado di adattarsi anche ai mobile devices) grazie alle varie media query presenti. 

Il sito è stato caricato su github pages.
